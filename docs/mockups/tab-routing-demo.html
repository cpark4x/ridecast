What Stays Visible

  • ✓ Tab bar on every screen
  • ✓ Mini player floats above tabs
  • ✓ Parent tab stays highlighted (Library on Source Detail, Discover on Topic)
  • ✓ Back button navigates within tab stack
  • ✓ Direct tab switching works from drill-down

Why href: null?

Without it, the screen either shows as a tab item (wrong — Source Detail shouldn't be a tab) or is inaccessible from within the tab stack.

With href:null, the screen is registered inside the tab navigator — preserving the persistent tab bar and mini player — but hidden from the tab strip.
