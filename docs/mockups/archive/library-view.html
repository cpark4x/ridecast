Upload is an action, not a tab. The FAB surfaces a bottom sheet for URL, PDF, EPUB, or TXT. Duration presets let users control depth before committing.
Library Design Decisions
Grouped by content, not version
Home flattens to episodes for playback. Library groups by source item — your content management view.
Upload-as-action FAB
Adding content is a transient action, not a persistent tab. Bottom sheet keeps context while the user configures their upload.
All states visible
Ready, Generating (pulsing), and Played states coexist. Played items are de-emphasized but not hidden — content has value beyond first listen.
